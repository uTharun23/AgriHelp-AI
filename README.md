# 🌱 AgriHelp-AI

AgriHelp-AI is a student-level AI-powered smart farming web application built using Flask. It helps farmers detect crop diseases, monitor soil conditions, analyze weather data, and get smart farming suggestions.

---

## Features

- Crop Disease Detection
- Weather Analytics
- Soil Monitoring
- AI Farming Assistant
- Smart Farming Dashboard
- Crop Health Monitoring

---

## Tech Stack

- Python
- Flask
- HTML, CSS, JavaScript
- OpenCV
- Weather API
